# 🎬 Netflix Data Cleaning & Visualization Project

## 📌 Project Overview

This project focuses on cleaning, preprocessing, analyzing, and visualizing the Netflix Movies & TV Shows dataset using Python.

The complete workflow includes data cleaning, feature engineering, exploratory data analysis (EDA), business insights, and dashboard creation to understand Netflix's content distribution and trends.

---

## 🎯 Objectives

- Clean and preprocess raw Netflix data
- Handle missing values and duplicate records
- Perform feature engineering
- Visualize important trends using Matplotlib and Seaborn
- Generate business insights through exploratory data analysis
- Build a dashboard summarizing key findings

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 💡 Key Insights

- Movies dominate Netflix's content library.
- The United States contributes the largest number of titles.
- Drama is the most common genre.
- Netflix experienced rapid content growth after 2015.
- Most titles are rated TV-MA and TV-14.
- Movie durations are generally between 80–120 minutes.
